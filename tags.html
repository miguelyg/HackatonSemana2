TAGS HTML5

  • FOOTER : Pie de pagina del elemento que lo contenga
  • NAV : Menu donde se encuentran los enlaces de navegacion
  • MAIN : Es la parte principal de la pagina, solo puede existir uno en el documento
  • ARTICLE : Es un contenido independiente
  • SECTION : Es una seccion del documento
  • FIGURE : Este tag se utiliza para colocar imagenes
  • FIGCAPTION : Es la leyenda de Figure
  • VIDEO : Este tag se utiliza para agregar un video
  • MARK : Es para colocar un texto resaltado
